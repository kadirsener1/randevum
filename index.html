⚙️ GitHub Bağlantı Ayarları

Sistem GitHub'a bağlanamadı. Lütfen ayarlarınızı kontrol edin.

Token Nasıl Alınır:
1. GitHub → Settings → Developer settings
2. Personal access tokens → Fine-grained tokens
3. Generate new token
4. Repository access: Repo'nuzu seçin
5. Permissions → Contents: Read and write
6. Generate token → Kopyalayın
